# 유희왕 주사위 게임 (YGO Dice Game)

유희왕 세계관의 두 가지 주사위 게임 — **Dice Duel**과 **Dungeon Dice Monsters** — 을 Flutter로 구현한 웹 게임 앱입니다.

---

## 게임 모드

### ⚔ Dice Duel (DD)
카드를 세트하고 주사위를 굴려 ATK를 계산, LP를 깎는 1:1 대결 게임.

- 매 라운드 몬스터를 세트 → 배틀 페이즈에서 양측 주사위 동시 굴림
- 주사위 결과에 따른 특수효과 (ATK ×2, 최소 보장, 저주 등 10종)
- 첫 턴 방어, 다음 라운드 예약 세트(preset), LP 0 게임오버
- **2인 대전** 또는 **vs CPU** 모드

### ♟ Dungeon Dice Monsters (DDM)
보드에 타일과 몬스터를 배치하고 이동·전투로 상대 로드를 격파하는 전략 게임.

- 주사위 3개 선택 → 크레스트(소환/이동/공격/마법/방어/트랩) 획득
- 레벨별 타일 모양(2~6칸)을 자유롭게 회전·배치
- BFS 기반 다단계 이동, 비행 유닛 지형 무시
- 전투 시 양측 주사위 굴림 + 몬스터 특수효과 적용
- 소환/이동 시 발동 특수효과 10종 (크레스트 보너스, 인접 피해, 위압 등)
- 마법(회복), 트랩(설치·발동) 크레스트 시스템
- **2인 대전** 또는 **vs CPU** 모드

## 기술 스택

| 항목 | 내용 |
|------|------|
| Framework | Flutter (Dart) |
| 상태관리 | flutter_bloc + equatable |
| 효과음 | audioplayers (WAV 프로그래매틱 생성) |

## 효과음

외부 오디오 파일 없이 **WAV를 Dart 코드로 프로그래매틱 생성**합니다.
`SfxService`가 오실레이터(sine/square/sawtooth) 기반으로 7종 효과음을 실시간 합성합니다.

- 주사위 굴리기, 이동, 소환, 전투, 피해, 승리, 마법, 트랩
- 화면 상단 🔊/🔇 버튼으로 음소거 토글
